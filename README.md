# PVOIL Login (Static Preview)

Simple static implementation of the PVOIL login page based on the Figma design.  
Intended for designers and developers to preview locally or on an internal network.

## Prerequisites

- Node.js 18+ installed on your machine.

## Install dependencies

From the project root:

```bash
npm install
```

## Run the dev server

```bash
npm run dev
```

By default this serves the folder at `http://localhost:3000/`.

- Open `http://localhost:3000/` in your browser.
- To let others on the same LAN see it, share:
  - Your IP from `ipconfig` (e.g. `192.168.1.23`)
  - The URL: `http://<your-ip>:3000/`
